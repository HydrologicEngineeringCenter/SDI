-d flags

The SDI allows for configuration of the user interface and computational properties through the use of -d flags.
-DSDI.MultiYearComputeOffsetEnabled=true -DSDI.HasMultiYearEvent=true skip validation flag...


Multi Year Event Flag

This flag allows the user to define multiple events per year, or an event that spans multiple years. Events are defined through the presence of null data in the data record for the primary location.
